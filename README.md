

# Albania Electricity Exports – Time Series Analysis

## Overview
This project analyzes Albania's quarterly electricity exports using time series methods in R. The study explores trend and seasonal components, tests stationarity, and compares forecasting models.

## Objectives
- Visualize the quarterly export data
- Identify trend and seasonal patterns
- Test stationarity using the Augmented Dickey-Fuller (ADF) test
- Apply exponential smoothing models (SES, Holt, Holt-Winters, ETS)
- Generate forecasts and evaluate model performance

## Methods Used
- Time series decomposition
- ADF stationarity test
- Exponential smoothing models
- Forecasting techniques

## Tools
- R
- forecast
- ggplot2
- tseries
- readxl
